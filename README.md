## Environment variables
This Django project is using these environment variables:
- DEBUG
- DB_NAME
- DB_USERNAME
- DB_PASSWORD
- SECRET_KEY
- RECAPTCHA_PUBLIC_KEY
- RECAPTCHA_PRIVATE_KEY
